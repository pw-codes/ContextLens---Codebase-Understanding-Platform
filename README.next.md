Modernized Next.js + TypeScript + Tailwind + React Flow scaffold

Quick start

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

App entry: [app/page.tsx](app/page.tsx#L1)
Layout: [app/layout.tsx](app/layout.tsx#L1)
Global styles: [app/globals.css](app/globals.css#L1)
React Flow demo: [components/CodebaseFlow.tsx](components/CodebaseFlow.tsx#L1)

Notes:

- This repository was scaffolded to demonstrate a minimal Next.js app that visualizes the existing static files using React Flow.
- Dependencies include `next@15`, `tailwindcss`, `reactflow`, and a placeholder shadcn UI package. Adjust versions as needed.
